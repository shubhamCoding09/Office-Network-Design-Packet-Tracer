# Office Network Design using VLAN, DHCP and Inter-VLAN Routing

## 📌 Project Overview
This project demonstrates a small office network with 3 departments (HR, IT, Sales) using VLAN segmentation, DHCP, and inter-VLAN routing.

## 🧠 Technologies Used
- Cisco Packet Tracer
- VLAN
- DHCP
- Inter-VLAN Routing

## 🏢 Network Design
- 3 VLANs:
  - VLAN 10 (HR)
  - VLAN 20 (IT)
  - VLAN 30 (Sales)

## ⚙️ Features
- Network segmentation using VLAN
- Automatic IP assignment using DHCP
- Communication between VLANs

## 🧪 Testing
- Successful ping between all VLANs
- DHCP working correctly
